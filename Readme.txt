			Calculator

  Что это
  -----------

  Очень простой калькулятор


  
  Документация и описание
  ------------------------

   Программа Calculator предназначена для нахождения и отображения в консоли суммы и 
  произведения двух вещественных чисел.
   Также программа предназначена для отработки практических занятий по темам лекций от 29.10.2021г.

   Программа поставляется в 5 вариантах, каждый из которых находится в отдельной папке.
   

  homework1   - файлы: Calculator.java, Add.java, Multi.java, Oper.java;  
  homework1_1 - файлы: Calculator.java;
  homework2   - файлы: Calculator.java;
  homework3   - файлы: Calculator.java;
  homework4   - файлы: Calculator.java, Oper.java;



  Состав пакетов
  --------------

  homework1_1 - написание простой программы с наличием пакета;
  homework1   - добавлен абстрактый класс;
  homework2   - добавлен внутренний класс;
  homework3   - добавлены анонимные классы;
  homework4   - добалены лябда-выражения.

  

  Установка
  ------------

   Перед запуском программы проверьте наличие установленной верии java
  JRE на вашем устройстве.
    
	
  
  Лицензия
  ---------

  Программа Calculator распространяется бесплатно вместе с исходным кодом.
